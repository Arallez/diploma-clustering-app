# Скрипты (разовые и утилиты)

Запуск из **корня проекта**: `python scripts/<имя>.py`

| Скрипт | Назначение |
|--------|------------|
| `fix_migrations.py` | Пометить миграции core.0003 и testing.0002 как применённые (если миграции уже в согласованном состоянии). |
| `fix_migrations.sql` | То же через SQL: `sqlite3 db.sqlite3 < scripts/fix_migrations.sql` |
| `init_tasks.py` | Создать базовые задания (Евклидово расстояние, центроид, ближайший кластер). |
| `init_materials.py` | Создать материал «Метод K-средних (K-Means)». |
| `fix_materials.py` | Обновить контент материала K-Means. |
